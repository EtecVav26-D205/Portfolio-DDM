Relatório do Projeto – Aplicativo de Localização do Allianz Parque

Projeto: Aplicativo de Abertura do Google Maps
Desenvolvido em: MIT App Inventor

O objetivo deste projeto foi desenvolver um aplicativo simples capaz de abrir automaticamente o Google Maps na localização do estádio Allianz Parque, do Palmeiras, utilizando a plataforma MIT App Inventor.

O aplicativo foi criado para demonstrar a integração entre aplicativos Android utilizando programação em blocos.

## 2. Funcionalidade do Aplicativo ## 

O aplicativo possui uma interface simples contendo:

Uma imagem do Allianz Parque;
Um botão chamado “Ver Localização”;
Um componente invisível chamado ActivityStarter.

Quando o usuário pressiona o botão, o aplicativo abre automaticamente o Google Maps mostrando a localização do estádio Allianz Parque.

## 3. Estrutura Utilizada ##
Componentes Visíveis
Componente	Função
Button1	Abrir o Google Maps
EscolheImagem1	Mostrar imagem do estádio
Componentes Invisíveis
Componente	Função
ActivityStarter1	Abrir o aplicativo Google Maps
## 4. Lógica de Programação ##

A lógica do aplicativo foi desenvolvida utilizando programação em blocos no MIT App Inventor.

O funcionamento ocorre da seguinte forma:

O usuário pressiona o botão;
O aplicativo envia a localização do estádio;
O ActivityStarter executa a ação;
O Google Maps é aberto automaticamente.
Estrutura dos blocos
quando Button1.Clique
    chamar ActivityStarter1.IniciarAtividade
## 5. Diferencial do Projeto ##

O diferencial deste projeto está na simplicidade e na integração com aplicativos nativos do Android.

Principais características:
Abertura automática do Google Maps;
Interface simples e intuitiva;
Utilização de programação em blocos;
Integração com aplicativos externos.
## 6. Tecnologias Utilizadas ##
MIT App Inventor;
Programação em blocos;
ActivityStarter;
Google Maps;
Android.
## 7. Conclusão ##

O desenvolvimento deste projeto permitiu compreender como integrar aplicativos externos utilizando o MIT App Inventor.

O aplicativo conseguiu abrir o Google Maps diretamente na localização do Allianz Parque de maneira simples e funcional.

Durante o desenvolvimento foram aplicados conhecimentos sobre:

Programação por eventos;
Componentes invisíveis;
Integração entre aplicativos;
Desenvolvimento mobile com App Inventor.

O projeto atingiu o objetivo proposto e pode ser melhorado futuramente com novas funcionalidades, como:

Adicionar rotas até o estádio;
Mostrar informações do local;
Inserir GPS em tempo real;
Abrir outros pontos turísticos automaticamente.
