O objetivo deste projeto foi desenvolver um aplicativo simples e funcional utilizando a plataforma MIT App Inventor, capaz de realizar o redirecionamento do usuário para um site externo de jogos esportivos através de um botão interativo.

O aplicativo foi criado com foco educacional e experimental, demonstrando como aplicativos mobile podem utilizar componentes de navegação para acessar conteúdos externos na internet.


## 2. Funcionalidade do Aplicativo ##

O aplicativo possui uma interface personalizada com temática esportiva do Palmeiras, contendo:

Imagem de fundo personalizada;
Botão “Ver Jogos”;
Componente invisível ActivityStarter;
Direcionamento automático para um link externo.

Quando o usuário clica no botão “Ver Jogos”, o aplicativo executa uma ação programada nos blocos:

quando Botão1.Clique
    chamar ActivityStarter1.IniciarAtividade

Essa ação utiliza o componente ActivityStarter para abrir um site externo no navegador do dispositivo.

## 3. Estrutura Utilizada ##
Componentes Visíveis
Componente	Função
Botão1	Acionar o redirecionamento
Imagem de Fundo	Personalização visual do app
Componentes Invisíveis
Componente	Função
ActivityStarter1	Abrir links externos

## 4. Diferencial do Projeto ##

O principal diferencial do aplicativo está na personalização visual e no tipo de redirecionamento realizado.

Enquanto muitos aplicativos escolares utilizam links para conteúdos educacionais ou institucionais, este projeto apresenta:

Fundo temático esportivo personalizado;
Identidade visual relacionada ao Palmeiras;
Redirecionamento para um site de jogos esportivos;
Interface simples e intuitiva.

Essa abordagem torna o aplicativo mais atrativo visualmente e demonstra como adaptar funcionalidades básicas para diferentes contextos e públicos.

## 5. Lógica de Funcionamento ##

A lógica do aplicativo é baseada em programação por blocos no App Inventor.

Fluxo do sistema:

O usuário abre o aplicativo;
A interface temática é exibida;
O usuário pressiona o botão “Ver Jogos”;
O componente ActivityStarter é ativado;
O navegador abre automaticamente o site esportivo configurado.
## 6. Tecnologias Utilizadas ##
MIT App Inventor
Programação em blocos;
Componente ActivityStarter;
Imagens personalizadas (.jpg);
Navegação via link externo.
## 7. Conclusão ##

O projeto demonstrou de forma prática como criar um aplicativo simples no App Inventor com capacidade de interação externa através de links.

Além da funcionalidade de redirecionamento, o aplicativo se destacou pela personalização visual esportiva, proporcionando uma experiência mais dinâmica ao usuário.

O desenvolvimento permitiu compreender conceitos importantes como:

Eventos de clique;
Programação em blocos;
Navegação entre aplicativos;
Utilização de componentes invisíveis;
Personalização de interface mobile.

O projeto atende ao objetivo proposto e pode ser expandido futuramente com novas telas, integração com APIs esportivas e atualização automática de jogos e resultados.
