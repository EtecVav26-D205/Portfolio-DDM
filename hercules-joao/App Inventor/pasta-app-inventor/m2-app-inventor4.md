O objetivo deste projeto foi desenvolver um aplicativo simples capaz de realizar uma ligação telefônica utilizando o MIT App Inventor.

O aplicativo foi inspirado na frase “Pintou notificação? É gol do Rony!” e possui apenas a função de ligar para o Rony ao clicar no botão.

## 2. Funcionalidade do Aplicativo ##

O aplicativo possui:

Uma imagem do Rony;
Um botão escrito “Ligue para o Rony!!!”;
Um componente de ligação telefônica.

Quando o usuário aperta o botão, o celular inicia automaticamente a ligação.

## 3. Estrutura Utilizada ##
Componentes Visíveis
Componente	Função
Button1	Fazer a ligação
Image1	Mostrar imagem do Rony
Componentes Invisíveis
Componente	Função
Ligação1	Realizar a chamada
## 4. Lógica de Programação ##

A programação foi feita utilizando blocos no MIT App Inventor.

Estrutura dos blocos
quando Button1.Clique
    chamar Ligação1.MakePhoneCallDirect

Quando o botão é clicado, o aplicativo realiza a ligação automaticamente.

## 5. Tecnologias Utilizadas ##
MIT App Inventor;
Programação em blocos;
Componente de ligação telefônica;
Android.
## 6. Conclusão ##

O projeto mostrou como criar um aplicativo simples de ligação utilizando o MIT App Inventor.

Durante o desenvolvimento foram utilizados conceitos de:

ligação divertida
humor automatico
top 5 melhores apps

O aplicativo funcionou corretamente realizando a chamada ao clicar no botão.
