## 🌎 Projeto 1

O objetivo do primeiro aplicativo é introduzir o uso do App Inventor, utilizando comandos básicos e fundamentais, como inserção de imagens, exibição de mensagens de texto e uso de botões.

### Desenvolvimento do Aplicativo

O aplicativo funciona com uma tela contendo uma imagem principal e um botão com o texto “Clique aqui”. Ao clicar nesse botão, é exibida a mensagem “Olá mundo”. Abaixo, há outro botão que permite fechar o aplicativo.

### Melhorias Realizadas

Em vez da imagem da Terra utilizada na apostila, foi inserida a imagem de um Opala vermelho. Além disso, foram alteradas as cores e o formato dos botões, deixando-os com aspecto arredondado.

## 🖌 Projeto 2

O objetivo do aplicativo é permitir que o usuário utilize diferentes cores para desenhar na tela.

### Desenvolvimento do Aplicativo

Ao abrir o aplicativo, o usuário encontra opções de cores e uma área em branco que simula um quadro. Após selecionar uma cor, é possível desenhar livremente na tela utilizando o dedo.

### Melhorias Realizadas

Foi alterada a imagem utilizada como área de desenho.

## 🎵 Projeto 3

O objetivo do aplicativo é possibilitar que o usuário interaja com uma imagem e ouça um som correspondente.

### Desenvolvimento do Aplicativo

Ao abrir o aplicativo, o usuário visualiza a imagem de uma cachoeira. Ao clicar sobre ela, é reproduzido o som de água corrente, simulando o som real de uma cachoeira.

### Melhorias Realizadas

Foram alterados tanto a imagem quanto o som utilizados no aplicativo.

## 📸 Projeto 4

O objetivo do aplicativo é permitir que o usuário utilize a câmera do dispositivo para tirar fotos.

### Desenvolvimento do Aplicativo

Ao abrir o aplicativo, o usuário vê um botão com o texto “Tirar foto”. Ao clicar nele, a câmera do celular é acionada, permitindo registrar uma imagem.

### Melhorias Realizadas

Foi alterada a cor do botão responsável por abrir a câmera.

## 🔥 Projeto 5
